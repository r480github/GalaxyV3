## Run Galaxy Locally üöÄ

##### Install Galaxy
```bash
git install https://github.com/r480github/GalaxyV3.git
cd GalaxyV3
```


<br>

##### Install dependencies

```bash
npm i  
```

##### Start

```bash
npm start
```
#### NOTE: The dafault port for Galaxy is 8000. You can visit it by typing localhost:8000/ into a web browser
